# 🚀 Trilha Rápida Flutter + Dart (Foco Prático)

## 🎯 Objetivo
Aprender rapidamente os principais conceitos:
- Widgets
- Stateless vs Stateful
- Future (async/await)
- Lifecycle
- Extensions

---

# 📆 Plano de Estudo (3 dias)

## 🟢 Dia 1 — Fundamentos de UI
### 📌 Widgets
- Tudo em Flutter é um Widget
- Exemplos: Text, Container, Row, Column, Scaffold

### 📌 Stateless vs Stateful
- Stateless: imutável
- Stateful: possui estado (setState)

### ✅ Prática
- Criar app contador simples

---

## 🟡 Dia 2 — Assíncrono
### 📌 Future + async/await
- Trabalhar com operações assíncronas
- Simular API

### 📌 FutureBuilder
- Atualizar UI com dados assíncronos

### ✅ Prática
- Criar tela que carrega dados fake

---

## 🔴 Dia 3 — Estrutura avançada
### 📌 Lifecycle
- initState()
- build()
- dispose()

### 📌 Extensions
- Criar helpers reutilizáveis

### ✅ Prática
- Refatorar app com boas práticas

---

# 🧪 Projeto Final
Criar um app de tarefas:

✅ Adicionar tarefa
✅ Listar tarefas
✅ Atualizar UI (Stateful)
✅ Simular carregamento (Future)
✅ Usar extension (formatar texto)
✅ Usar lifecycle (logs)

# 💡 Dicas rápidas
- setState() = re-render
- Future = Promise (Node.js)
- Extensions = helpers
- Stateless = performance
- Stateful = interação
